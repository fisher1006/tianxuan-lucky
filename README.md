# 小红书桃花运工具站 - 需求文档

## 项目概述

简单工具站，卖网站地址+邀请码，几毛钱一个走量，主要靠小红书运营赚钱。

---

## 变现路径

小红书发图文笔记 → 笔记挂商品链接（卖网站地址/邀请码） → 用户购买后人工发货（私信发链接+邀请码） → 用户到工具站输入邀请码使用

---

## 功能清单

### 1. 脱单抽签
用户输入基本信息后，随机抽取脱单运势签文，提供简单指引。

### 2. 今日桃花运
每日桃花运测试，输入基本信息后输出今日桃花指数和简单建议。

### 3. 相亲匹配度
输入自己和相亲对象的基本信息，计算匹配度分数并给出简短评语。

### 4. 复合抽签
针对想要复合的用户，提供抽签占卜服务，给出简单指引。

---

## 商品设计

| 商品名称 | 价格 |
|---------|------|
| 单次邀请码 | ¥0.5 / ¥0.88 / ¥1.0 |
| 批发装（10次） | ¥6.6 |

---

## 技术需求

- 简单工具站，4个独立页面
- 邀请码验证功能（输入邀请码解锁使用）
- 不需要AI、数据库、会员系统
